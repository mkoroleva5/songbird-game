# Song Bird game for RS School

## Deployment  

- [Song Bird game](https://mkoroleva5-songbird.netlify.app/)

## Run Locally

Clone the project

```bash
  git clone https://github.com/mkoroleva5/songbird-game.git
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```
